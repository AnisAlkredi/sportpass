# SportPass Landing (Next.js)

صفحة هبوط تسويقية مبنية بـ Next.js.

## التشغيل محليًا

```bash
cd landing-next
npm install
npm run dev
```

افتح:

`http://localhost:3000`

## النشر على Vercel

1. اربط نفس الريبو في Vercel.
2. اختر `Root Directory` = `landing-next`.
3. Build Command = `npm run build`.
4. Start Command = `npm run start` (أو اتركها افتراضيًا).

سيتم نشر صفحة SportPass مباشرة.
